# 🍕 Pizza Sales Performance Dashboard

## 📊 Project Overview

This project analyzes pizza sales data using Microsoft Excel.
It provides insights into delivery performance, product popularity, and traffic impact.

---

## 🔍 Key Insights

* Majority of orders are delivered on time
* Non-Veg and Veg pizzas are the most popular
* High traffic increases delivery delays

---

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Data Analysis
